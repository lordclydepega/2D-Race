# 2D Race

## 📝 Game Description
**2D Race** is a high-performance, top-down racing simulation built with vanilla JavaScript and HTML5 Canvas. It features a "Grand Circuit"—a vast, scrolling 3000x2000 pixel map that challenges players to master high-speed cornering and reflex-based obstacle avoidance. 

The game emphasizes precision and speed, requiring players to navigate a track bordered by physical boundaries. Whether dodging oscillating hazards or maintaining the perfect racing line to avoid speed-dampening terrain, every millisecond counts toward reaching the top of the local Hall of Fame.

## ⚙️ Setup Instructions
To run the code locally on your machine, follow these steps:

1.  **Clone or Download**: Download the project files and ensure the `index.html` file is in your desired folder.
2.  **Open with Browser**: 
    *   Right-click on `index.html`.
    *   Select **"Open With"** and choose a modern web browser (Google Chrome, Microsoft Edge, or OperaGx).
3.  **Alternative (Live Server)**: If you are using VS Code, you can right-click `index.html` and select **"Open with Live Server"** for an optimized experience.
4.  **No Installation Required**: Since this is built with vanilla web technologies, no external libraries or dependencies are needed to play.

## 🕹️ List of Controls
| Key | Action |
|-----|--------|
| **Arrow Up** | Accelerate |
| **Arrow Down** | Brake / Reverse |
| **Arrow Left** | Steering (Turn Left) |
| **Arrow Right** | Steering (Turn Right) |
| **ESC** | Return to Main Menu |
